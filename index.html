NETWORK SIM

Link to GitHub
(Network Simulation in Cisco Packet Tracer) Modeling an attendance system where students can log attendance records via a centralized HTTP server. Key Components: - Router 1: Routes internal network traffic - Switch: Connects all local devices (PCs, Admins, Server) - Server 0: Hosts the attendance web service - PCs: Represent student and staff clients accessing the system - Admin PC: Has privileged access for record management
